# 🎨 FreeCodeCamp CSS Visual Projects

you can see the live version of this project here:

---

## 🐧 Project 1: Flappy Penguin
An advanced CSS illustration demonstrating complex shapes, positioning, and interactive transitions.



### ✨ Key Features:
* **Custom CSS Shapes:** Built entirely using `div` elements with varied `border-radius` and `transform` properties.
* **Interactive Transitions:** Uses `transition: transform 1s ease-in-out` to scale the penguin when clicked (`:active`).
* **Dynamic Animations:** Implements `@keyframes` for a "waving" wing effect and a parallax-style mountain background.
* **Variable Driven:** Utilizes CSS Variables (`--penguin-skin`, etc.) for easy theme management.


## 🎡 Project 2: Ferris Wheel
An exploration of synchronized CSS animations and rotational physics.



### ✨ Key Features:
* **Infinite Animation:** Uses `animation-iteration-count: infinite` for a seamless mechanical rotation.
* **Counter-Rotation Logic:** Implemented a secondary animation on the "cabins" to keep them upright while the main wheel rotates — a classic lesson in nested transforms.
* **Timing Functions:** Used `linear` timing to maintain a constant, realistic speed for the wheel.

---

## 🛠 Tech Stack Used
* **HTML5:** Semantic structure and deep nesting for hierarchical animations.
* **CSS3:** Flexbox, Absolute Positioning, Keyframes, and 2D/3D Transforms.
* **Logic:** Applied CSS math with `calc()` for responsive ground positioning.
