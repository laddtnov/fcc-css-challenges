# CSS Animated Ferris Wheel 🎡

A fun and visually engaging project built to master **CSS Animations** and **2D Transformations**. This project is part of the freeCodeCamp "Responsive Web Design" curriculum.

## ✨ Project Highlights
* **Pure CSS:** No JavaScript or external libraries used.
* **Complex Keyframes:** Synchronized animations to keep the cabins upright while the wheel rotates.
* **Transformations:** Extensive use of `transform`, `transform-origin`, and `rotate`.
* **Responsive:** Designed to scale smoothly across different screen sizes.

## 🛠️ Concepts Learned
* Creating custom animations with `@keyframes`.
* Managing animation timing with `animation-timing-function`.
* Balancing nested animations (counter-rotating elements).
* Absolute and relative positioning within a circular layout.
